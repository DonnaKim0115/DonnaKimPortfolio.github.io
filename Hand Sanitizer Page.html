Hand Sanitizer Dispenser

What?

An automated hand sanitizer machine! Made using Arduino, an ultrasonic sensor, a stepper motor, and a driver board.



How Does it Work?

The ultrasonic sensor detects when an object is below the nozzle of the hand sanitizer bottle and when an object is within 4cm, the motor will activate to pull the nozzle down and dispense the sanitizer.
